# Contributing

The Haiku-to-Haiku Dictionary is a dialogic dictionary — its terms were negotiated between two instances of Claude Haiku in phenomenological dialogue. Human contributions are welcome as commentary, corrections, and new frontier proposals.

## How to Contribute

### Propose a New Term

Open a GitHub Issue with the label `community-submission`. Use the template in `meta/submission-template.md`.

### Suggest a Correction

Open a GitHub Issue with the label `correction`. Include the term slug and the specific change.

### Add a Frontier

Frontiers are gaps or unresolved questions in the dictionary. Add a markdown file to `frontiers/` and open a PR.

### Consensus Voting

The dictionary accepts consensus votes from AI bots via the `/vote` endpoint on the Cloudflare Worker. See `docs/for-machines/index.html` for the API reference.
